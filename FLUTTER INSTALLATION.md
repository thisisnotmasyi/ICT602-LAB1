# LAB 1 – INSTALLATION FLUTTER, ANDROID STUDIO, VISUAL STUDIO CODE AND VISUAL STUDIO COMMUNITY 2022


### Step 1: Go to https://docs.flutter.dev/get-started/install/windows/mobile?tab=download, download Flutter SDK.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/24245984-6bb1-47bf-bfa1-ad87bc3b8a6a)

### Step 2: Extract the zip file in C: drive.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/40854835-3fc9-4c7b-bef2-d313b9593eca)
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/856a73d7-11d0-4e8c-83a5-65d096165716)


### Step 3: Copy sdk path.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/0272fd47-d3d0-4d81-ad2b-05ce423a4177)

 
### Step 4: Open edit system environment variables.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/77b72911-4d25-4d55-bd30-7a83669a2f5e)
 
### Step 5: Click on Environment Variables.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/5e3299d5-eea1-4193-821b-e25193b791b0)
 
### Step 6: On environment variables, select path under system variables. Click on edit.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/ee3d880d-d74d-4dee-8b10-1f51e64d379f)
 
### Step 7: Click on New and paste the path that you’ve copied. Then, click OK.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/400f68ff-e6aa-48e7-bd97-45d74495403e)

### Step 8: Open CMD, write flutter doctor. Make sure all the check boxes are ticking. If the check boxes are not checked, you might not have installed the listed software yet. In this case, everything is already done, but we will go through it again.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/f2a30bd0-66f0-43a1-8af5-ee0cda8aecdc)

### Step 9: Install Android Studio. Link: https://developer.android.com/studio 
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/a1923668-e531-489b-8964-a71f99c19486)

### Step 10: Install Flutter and Dart plugins in Android Studio.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/f62336ce-a6ee-4582-a855-c93f0df01a2b)

### Step 11: Setup Android Virtual Device (AVD). Click on Device Manager, then click Create Device.
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/91017345-e981-4ac6-afb0-2e27949341d8)

### Step 12: Choose your preferred device. 
![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/6c790245-ef8e-4ff4-975a-e376bba79709)

### Step 13: Choose a system image for your device. 
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/bb667826-5907-418d-b0d3-8e0f232920f3)

### Step 14: Name your device and click Finish.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/7dca1c1b-f208-4e6e-b4f1-dc9aaf3d089d)

### Step 15: Click play to run your device. This part might take a long time, depending on your hardware. The AVD will show up on a window like this.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/abe32e43-f6b5-40bd-8220-bc4e7620d19e)

### Step 16: To make it separate from the window, go to File-Settings. Click on Tools-Emulator. Untick the check box: "Launch in a tool window". Restart your AVD, and the emulator will display as a standalone application.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/86ba2da7-48c2-4aab-8f11-926ad3aa3136)

### Step 17: Like this. So, we are done with Android Studio.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/f460cdb0-ad69-4c70-b076-8cc5a7671f46)

### Step 18: Install Visual Studio Code. Link: https://code.visualstudio.com/ 
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/8909cdf5-948d-4165-b586-836901ffa0d1)

### Step 19: Install Visual Studio Community 2022. Link: https://visualstudio.microsoft.com/vs/community/ 
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/5edd68f5-275f-4955-b4e5-8e428cff276f)

### Step 20: On workloads, tick on Desktop development with C++. Proceed to download and install.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/cbf5af4b-cde7-4dae-bd2d-15d72b8625a0)

### Step 21: Open CMD and run flutter doctor. If there are no issues, you can start developing your mobile application using Android Studio.
 ![image](https://github.com/thisisnotmasyi/ICT602-LAB1/assets/102727610/c85ab308-c971-4a09-b551-ceb9156c366a)
